My First Bootstrap Page!

HTML

HyperText Markup Language

It is a markup language used for making the baseline structure of a webpage. It tells the browser what elements should be present during the initial rendering of the page and how to format them. It is the standard language used in most webpages.

CSS

Cascading Style Sheets

A styling language that works alongside markup languages like HTML. It is used to add pretty colors and offers a more complex, but flexible method of formatting webpages. CSS styling can be coded into an HTML document, either in the head tag or in-line via the style attribute. However, CSS stylesheets are most commonly a separate file linked through the link tag.

Bootsrap

Bootstrap Version 4

A front-end framework used to make responsive webpages a whole lot easier to code. It allows a developer to quickly format elements to fit numerous devices. It offers styling and formatting templates that make the webpage compatible with devices that may have different dimensions such as phones and tablets.
